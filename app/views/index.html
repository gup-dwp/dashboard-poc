DWP design patterns

These patterns for DWP services are based on the research and experience of the DWP design community. You should use them to build on the GOV.UK design patterns.

These are the recommended standards for DWP service design. You should use them even when live services (like Carer’s Allowance) don’t yet follow them.

More guidance

Visual style

There are 2 elements guides which give details of visual style.

If you’re working on a citizen-facing service that’s hosted on GOV.UK, use the GOV.UK elements guide.

If you’re working on a service that isn’t hosted on GOV.UK or a service for DWP staff, use the DWP elements guide.

Content style

Use the GOV.UK style guide as a reference for content style, spelling and grammar conventions for any service you’re working on.
